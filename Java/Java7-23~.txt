7-23 다형성(polymorphism)
- 여러가지 형태를 가질 수 있는 능력
- 조상 타입 참조 변수로 자손 타입 객체를 다루는 것


조상타입의 참조 변수 t와 자손타입의 SmartTv객체가 타입이 불일치해도 조상과 자손의 관계면 사용가능
이미지 1



- 객체와 참조변수의 타입이 일치할 때와 일치하지 않을 때의 차이?

일치할 때 멤버변수가 7개면 모든 기능을 사용할 수 있음
일치하지 않을 때 자손타입의 멤버변수가 7개면 조상타입에 정의된 멤버변수만 사용할 수 있음
*참조변수로 사용할 수 있는 멤버의 갯수가 달라짐


- 자손 타입의 참조변수로 조상 타입의 객체를 가리킬 수 없음

객체가 조상이 되면 에러가 남
실제 사용할 수 있는 기능보다 리모컨의 기능 개수가 많으면 안됨

7-26 instanceof 연산자
- 주로 조건문에 사용.
- 참조변수의 형변환 가능여부 확인에 사용. 가능하면 boolean 값인 true 반환.
- 형변환 전에 반드시 instanceof로 확인해야 함.
- 참조변수(리모콘)을 변경함으로써 사용할 수 있는 멤버의 갯수를 조절 가능.
- 참조변수 instanceof 타입(클래스 명) 형태로 사용.
